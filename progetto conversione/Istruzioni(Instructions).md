#Blocca il server dal download dei file

Per impedire allo scraper di copiare il tuo sito Web con HTTrack e altre fotocopiatrici di siti Web, dovrai inserire il file .htaccess nel tuo Public_html (in un altro dove la tua cartella index.html O index.php) L'ho appena fatto qui come puoi vedere
la mia cartella index.html e .htaccess si trova nello stesso livello di cartella. inoltre se non riesci a vedere la cartella .htaccess significa che l'ho nascosta poiché tutti i file iniziano con il .(punto) per mostrare questo video https://www.youtube.com/watch?v=SLvHzL12028

#Strumento di sviluppo disabilitato

Per disattivare lo strumento per sviluppatori, dovrai inserire questo script appena prima di chiudere il tag body, proprio come ho fatto in index.html

<!-- Questo tag Script disabiliterà lo strumento per sviluppatori -->
<script disable-devtool-auto
  src='https://cdn.jsdelivr.net/npm/disable-devtool/disable-devtool.min.js'></script>
